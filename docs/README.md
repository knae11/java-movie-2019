# 영화예매

## 구현할 기능 목록
- 현재 상영중인 영화를 보여준다.
- 사용자로부터 예약할 영화를 입력 받는다. ( 존재하는 영화의 번호가 아닌 경우 예외처리 )
- 예약할 영화의 시간표를 출력한다. 
- 예약할 시간을 입력받는다. ( 존재하는 상영시간인지, 현재시간 이후인지 확인, 현재와 1시간 이내 차이인지 )
- 예약할 인원을 입력받는다. (1이상의 정수인지, 예약가능인원보다 크지 않은지 확인)
- 1: 예약종료, 2: 추가예약 여부를 입력받는다. (1,2 이외 예외처리)
- 1 선택시 : 사용자로부터 포인트 사용금액을 입력받는다. (0 이상 ~ 영화값 이하 정수)
- 2 선택시 : 처음으로 돌아가 반복한다.
- 사용자로부터 결제 방법을 입력 받는다.(1:카드, 2:현금)
- 결제 금액을 계산한다. (적립 포인트 적용, 카드 결제 시 5% 할인, 현금 결제 시 2% 할인)
- 결제 금액과 안내멘트를 출력한다.

## 출력
- 선택에 도움이 되는 가이드 문구는 `##`을 붙인다.
- 예외상황은 `[ERROR]`를 붙인다.
